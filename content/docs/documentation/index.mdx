You can find the npm package by the name  [md-to-jsonify](https://npmjs.com/md-to-jsonify)

## Features

It comes with the following features out of the box:

1. Write content using MDX.
2. Transform and validate content.
3. Optimize the iteration for converting the data.
4. Split out link in separate data feilds with ```link``` property.
5. Passed multiple edgecase for symbols that does have a meaning on markdown like ```[]``` , ```()```.
6. Numbering the links - ```linkX``` , while X being the number of links with a given row.
7. More is comming like - parsing the convetable data's from the entire md file 

## How is it built

We have used Javascript  , Typescript as well some utility functions.
